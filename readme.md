# tevere (Te veré)

A script overhaul to yt-dlp's tver.jp downloader.

## Why this exists?

Because the template expansion of yt-dlp don't work as expected with
tver.jp

## What this script does?

Fetches data from the tver public API and extract the right values,
assembles a better filename and passes it to yt-dlp.

It also saves the API episode data to disk.
